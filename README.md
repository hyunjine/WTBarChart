# Including in your project
**setting.gradle**
```java
allprojects {
    repositories {
        ...
        maven { url 'https://jitpack.io' }
    }
}
```

**build.gradle(Module)**
```java
dependencies {
    ...
    implementation 'com.github.hyunjine.WTBarChart:wtbarchart:0.0.3'
}
```
